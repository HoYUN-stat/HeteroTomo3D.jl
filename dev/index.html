HeteroTomo3D.UnitQuaternion — Type
UnitQuaternion{T<:Real}

Represents a unit quaternion q = (ω, x, y, z) ∈ S^3 for singularity-free 3D rotations.

source
Base.:* — Method
*(q1::UnitQuaternion, q2::UnitQuaternion)

Overloads the multiplication operator to compose two rotations: R{q1} R{q2} = R_{q1 q2}.

source
Base.conj — Method
conj(q::UnitQuaternion)
inv(q::UnitQuaternion)

For a unit quaternion, the inverse is equal to its conjugate (ω, -x, -y, -z).

source
HeteroTomo3D.planar_rotation — Method
planar_rotation(q::UnitQuaternion)

Extracts the first column of the 2D in-plane rotation matrix (SO(2) component), i.e., it returns a Tuple (c, s) representing [c -s; s c] ∈ SO(2).

source
HeteroTomo3D.projection_axis — Method
projection_axis(q::UnitQuaternion)

Extracts the projection axis rq = R{q}^{-1} e_3 ∈ S^2. Returns a Tuple (x, y, z) to guarantee 0 heap allocations.

source
HeteroTomo3D.shortest_arc — Method
shortest_arc(ux::Real, uy::Real, uz::Real)

Computes the shortest-arc rotation E(u) ∈ SO(3) mapping e3 to u ∈ S^2. Requires u ∈ S^2 - {-e3}.
